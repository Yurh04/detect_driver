# 测试模型说明

## 当前使用的测试模型

**模型**: YOLOv8n COCO预训练模型  
**类别数**: 80类  
**位置**: `backend/models/best.pt`

## 重要说明

⚠️ **这不是您训练的驾驶员行为检测模型！**

当前使用的是YOLOv8n的COCO数据集预训练模型，用于测试系统功能是否正常。此模型可以检测80种日常物体，包括：
- `person` (人)
- `cell phone` (手机) 
- `cup` (杯子)
- `car` (汽车)
- 等等...

## 测试建议

### 可以测试的功能：
✅ 系统是否能正常启动  
✅ 视频上传功能是否正常  
✅ 实时摄像头检测是否工作  
✅ 检测框绘制是否正确  
✅ 统计功能是否正常

### 检测效果：
- ✅ 可以检测到人、手机、杯子等日常物体
- ❌ 不会特别针对"抽烟"、"用手机"、"喝水"这三种驾驶员行为优化
- ❌ 检测结果可能不如专门训练的模型准确

## 切换到您的训练模型

完成系统测试后，要使用您训练的模型：

1. 从同学那里获取训练好的 `best.pt` 文件
2. 备份当前测试模型（可选）：
   ```bash
   mv backend/models/best.pt backend/models/yolov8n_coco.pt
   ```
3. 将训练好的模型复制到：
   ```bash
   cp /path/to/your/best.pt backend/models/best.pt
   ```
4. 重启后端服务

系统会自动检测模型的类别数和名称，无需修改代码！

## 验证模型信息

启动后端后，检查日志输出：
```
INFO:detection:检测到 3 个类别: ['Smoke', 'Phone', 'Drink']
```

如果看到3个类别且名称正确，说明您的训练模型已正确加载！
